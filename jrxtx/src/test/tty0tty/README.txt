  - To run:
         ./tty0tty   
         
         (/dev/pts/1) <=> (/dev/pts/2) 
         
  - Or, with fix aliases, run:       
         ./tty0tty /dev/ttyS99 /dev/ttyS100 
        
         (/dev/ttyS99) <=> (/dev/ttyS100) 

  the connection is (no handshaking!):
  
  TX -> RX
  RX <- TX
